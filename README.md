# LLM-Assisted Security Requirements Engineering Evaluation

This repository contains the replication materials for our study on LLM-assisted asset identification and STRIDE threat mapping using software requirements artifacts from the iTrust2 and CULRepo datasets.

The repository includes the prompts used in the study, manually curated reference assets and threat mappings, generated outputs from the evaluated LLMs, and repeated runs used for the output stability analysis.

## Repository Structure
- prompts/ 
- generated_outputs/ 
- manual_reference_sets/ 
- output_stability/ 
- dataset_sources/ 

## Prompts

The `prompts/` folder contains the prompt templates used in the evaluated configurations, including both single-shot and staged prompting approaches.

## Generated Outputs

The `generated_outputs/` folder contains the assets, threat mappings, and related outputs generated by the evaluated LLMs across different datasets and prompting configurations.

The evaluated models include:

- GPT-5.5
- Claude Opus 4.7
- DeepSeek-V4

## Manual Reference Sets

The `manual_reference_sets/` folder contains the manually curated reference assets and STRIDE threat mappings used during evaluation.

The reference sets were created through manual analysis of the software artifacts and refined through iterative review and semantic normalization during evaluation.

## Output Stability

The `output_stability/` folder contains repeated executions used for the representative output stability analysis discussed in the paper.

## Datasets

The study uses the following datasets:

- iTrust2
- CULRepo

To avoid redistribution issues, this repository include dataset references instead of the complete original datasets.

## Notes

Some generated outputs differed from the manually curated reference sets in wording, abstraction granularity, and operational decomposition. As discussed in the paper, semantic normalization and contextual analysis were used during evaluation to determine meaningful correspondence between generated and manually curated artifacts.

This repository is intended to support transparency and reproducibility of the reported study.
